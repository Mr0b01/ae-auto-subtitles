# AED Subtitles v1.0.135

AED Subtitles is a Descript-style subtitle workflow for Adobe After Effects: local Whisper transcription, caption review, reference text correction, style preview, and native AE layer generation.

## Download

Download and run:

`AE-Auto-Subtitles-Installer-1.0.135.pkg`

## What changed

- Added a real GitHub product page.
  - The README now explains what the extension does, why it exists, how to install it, how to run it, and how to troubleshoot common AE/Whisper problems.

- Added direct installer links.
  - The README and installation guide link to the latest release asset using GitHub's `releases/latest/download/...` path.

- Documented the current stable defaults.
  - Default model: `turbo`
  - Default audio source: `Active comp mix`
  - Default preset: `Bold: Yellow highlight`
  - Default layout controls: size 48 px, max chars 42, block width 900 px, block scale 91 percent, offset Y +224 px

- Clarified the stable subtitle workflow.
  - Scan or Rescan the active comp.
  - Wait for Whisper Turbo or press Retiming.
  - Review captions.
  - Add reference text when needed.
  - Run or Apply Changed.

- Clarified the current backplate status.
  - Dark backplate / Hug Lines styles are not treated as a stable default renderer path in this release.

## Installer

The package installs the CEP extension into the current user's Adobe CEP extensions folder and prepares the local Python runtime dependencies.

The installer includes:

- CEP panel files
- Python backend scripts
- ExtendScript renderer
- offline Python wheels
- bundled ffmpeg
- cached Whisper Turbo model when available

If the Turbo model is not bundled, the first transcription may download it automatically.

## Requirements

- macOS
- Adobe After Effects with CEP/ExtendScript support
- Python 3 available on the machine
- AE preference enabled: `Allow Scripts to Write Files and Access Network`
- Internet on first Turbo run if the model is not bundled or cached
