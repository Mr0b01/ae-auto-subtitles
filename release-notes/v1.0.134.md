# AED Subtitles v1.0.134

Descript-style subtitle workflow for Adobe After Effects with local Whisper transcription, caption review, reference text correction, style preview, and AE layer generation.

## Highlights

- Fixed skipped audio in montage-style timelines.
  - File-backed audio layers are now all selected by default instead of only the two longest layers.
  - This prevents subtitle gaps when voice is split across many edited clips.

- Whisper Turbo is now the default recognition model.
  - `small`, `medium`, and `large-v3` remain available as manual fallbacks.

- Added changed-only reference apply.
  - `Apply Changed` updates only captions whose reference text differs from the model text.
  - Existing unchanged subtitle layers are left untouched.

- Improved native QA.
  - The panel can run syntax checks, unit smoke tests, real AE layout self-test, and real AE timing self-test.

## Installer

Download and run:

`AE-Auto-Subtitles-Installer-1.0.134.pkg`

The installer places the CEP extension into the current user's Adobe CEP extensions folder and prepares the local Python runtime dependencies.

## Requirements

- macOS
- Adobe After Effects with CEP/ExtendScript support
- Allow Scripts to Write Files and Access Network enabled in AE preferences
- Python 3 available on the machine
- Internet may be required on first run if the turbo Whisper model is not bundled in the installer cache
